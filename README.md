# Binary to Integer
###### This code takes any sized binary number as a string and converts it to its integer equivalent. 

## For Example:
###### a="0000", b="0001" -> int 1
###### a="1110", b="0101" -> int 19
###### a="0011", b="1001" -> int 12
###### a="101111", b="1111" -> int 62
###### a="101010101", b="111111" -> 404
